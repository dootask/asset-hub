# Asset Hub 资产与耗材管理插件

Asset Hub 是一个面向企业的资产与耗材全生命周期管理插件，可以无缝集成到 DooTask 中，帮助你在同一套协作平台里完成资产台账、流转审批、盘点与统计分析。

## ✨ 插件亮点

- **资产全生命周期管理**：覆盖采购、入库、领用、借用 / 归还、派发、维修、报废、回收、遗失等全流程，自动沉淀操作记录。
- **耗材精细化管控**：支持耗材台账、出入库记录、最低库存预警、盘点与告警报表，避免库存失控。
- **可视化仪表盘**：首页展示资产规模、分布、价值趋势等关键指标，以及待办审批与常用入口。
- **内置审批流转**：围绕资产操作内嵌审批能力，支持草稿、审批中、已通过、已拒绝、已取消等状态。
- **多维统计报表**：按公司、部门、分类、状态等维度统计资产与耗材情况，支持导出和审计追踪。
- **深度集成 DooTask**：继承 DooTask 的用户、权限和主题，无需二次登录，配合任务 / 待办实现一站式协作。

## 🚀 你可以用 Asset Hub 做什么？

- 建立企业统一的资产与耗材台账，告别分散的 Excel 表格。
- 规范资产领用、借用、归还与报废流程，减少丢失和责任不清。
- 通过审批流程和操作日志提升资产管理的透明度与可追溯性。
- 定期盘点资产与耗材，掌握真实库存与闲置情况。
- 通过报表洞察资产投入产出、部门占用与生命周期成本。

## 👥 典型使用角色

- **系统 / 资产管理员**
  - 维护公司与组织架构
  - 配置资产分类、字段、状态与操作类型
  - 处理资产采购、入库、调拨、报废等操作
  - 管理盘点任务与报表导出

- **审批人（部门主管 / 财务 / 资产负责人等）**
  - 审批采购、入库、领用、借用 / 归还等申请
  - 在审批中心查看待办与历史记录

- **普通员工 / 申请人**
  - 发起资产领用 / 借用 / 归还等申请
  - 查看自己名下的资产与相关操作记录

实际可使用的功能范围由系统管理员在 DooTask 中进行权限配置。

## 📥 如何在 DooTask 中打开插件？

1. 确认你的工作区已由系统管理员在 DooTask 应用市场或后台安装并启用「Asset Hub 资产管理」插件。
2. 在 DooTask 中，打开左侧菜单或应用中心，找到 **Asset Hub 资产管理** 入口。
3. 点击后即可进入 Asset Hub 首页，系统会自动使用你当前的登录身份、语言和主题，无需再次登录。

如打开页面提示无权限，请联系工作区管理员为你分配 Asset Hub 相关权限或角色。

## 🧭 核心功能导航

### 首页 / 仪表盘

- 资产与耗材概览（数量、价值、状态分布等）
- 待办审批与常用操作快捷入口
- 帮助中心与使用指南入口

### 系统管理

- **公司与组织管理**：管理公司主体、部门结构等基础信息。
- **角色与权限**：配置系统管理员、资产管理员、审批人等角色及权限范围。
- **审批配置**：为不同类型的资产操作配置是否需要审批以及对应流程。
- **数据报表与自定义报表（规划中）**：集中管理统计报表与导出能力。

### 资产管理

- **资产设置**：自定义资产字段、状态与分类，满足不同企业的管理习惯。
- **资产列表**：按多条件筛选、排序和分页查看资产台账。
- **导入 / 导出**：支持批量导入历史资产数据，并导出为报表。
- **资产操作流程**：
  - 采购与入库
  - 领用、借用 / 归还、派发
  - 维修、报废、回收、遗失等其他操作
- **资产盘点**：创建盘点任务，记录盘盈盘亏情况。

### 耗材管理

- **耗材台账**：维护各类耗材的编码、名称、规格、库存等信息。
- **出入库管理**：记录领用、补货、退库等操作，支持批量操作。
- **库存预警与告警**：设置最低库存阈值，及时发现缺货风险。
- **盘点与审计报表**：支持耗材盘点、变动明细与审计记录导出。

### 帮助中心

- 查看使用指南、常见问题与最佳实践示例。
- 了解资产与耗材管理相关的操作建议。

## 🔐 权限与数据安全

- Asset Hub 基于 DooTask 的账号与权限体系运行，不需要单独的账号体系。
- 不同角色在界面中看到的菜单与操作权限会有所不同，具体取决于管理员配置。
- 所有关键操作都会记录到资产操作与审批记录中，用于审计与追溯。

## ❓ 常见问题

- **Q：打开后提示没有权限怎么办？**  
  A：请联系你的工作区管理员，确认已经为你分配了 Asset Hub 插件的访问权限以及相应的角色（如资产管理员、审批人或普通使用者）。

- **Q：Asset Hub 中的数据会影响 DooTask 其他模块吗？**  
  A：Asset Hub 主要管理资产与耗材数据，并通过审批、待办等方式与 DooTask 进行集成。是否联动其他模块（如任务、流程等）取决于管理员的整体配置策略。

- **Q：如何学习完整的使用方法？**  
  A：建议从 Asset Hub 首页进入「帮助中心」，查看官方示例与推荐流程；也可以在企业内部约定资产管理制度并结合本插件使用。
